    • Create React App

    What is Create React app?

    Create react app is a command line tool that helps us generate a boilerplate react Application, with according webpack configuration to make developers life easier

    npm i create-react-app
    create-react-app yourappname

    • JSX

    JSX is the language used by react to allows programmers towrite xml like (close to html) tags and renders them to javascript

    This allows developers to easily write their code while adapting to a minimal amount of difference. While also providing a very dynamic way of programming.
    Example:

    render( <li>I love react</li>)
  • Every JSX element return will be converted into the according document.createElement()
    • Component types

    In React there are two types of components

    Class Based and Functional components

    Class Based: is a class that return a component

    class Welcome extends React.Component {
    
            render() {
    
              return <h1>Hello, {this.props.name}</h1>;
    
            }
          }

    Functional: is a function that return a component

    function Welcome(props) {
            return <h1>Hello, {props.name}</h1>;
          }
  • The most recent and popular approach to creating a react component is using functions.
